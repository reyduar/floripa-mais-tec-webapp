Calculadora

O projeto pretende criar uma calculadora com operações matemáticas básicas utilizando a linguagem de programação JavaScript.

  • Instruções de Uso

  • 1. Insira o primeiro valor no campo de texto chamado Valor#1.
  • 2. Insira o segundo valor no campo de texto chamado Valor#2.
  • 3. Selecione a operação matemática que deseja que a calculadora execute.
  • 4. Pressione o botão calcular e o resultado é exibido no campo de texto chamado resultado.

  • Calculadora ECMAScript v1.0

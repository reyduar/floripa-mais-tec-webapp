Distribuição dos convidados em setores

O programa busca o nome do setor para cada grupo de idade, inserindo a idade do convidado, o nome de cada setor é diferenciado por cores.

  • Instruções de Uso

  • 1. Insira uma idade válida no campo de texto chamado Idade do convidado.
  • 2. As idades aceitas são 13+ anos.
  • 3. Pressione o botão Buscar Setor e o resultado é exibido no campo de texto chamado resultado.
